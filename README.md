# Project Contents

This repository contains a modern, responsive portfolio website for Tushar Gupta, built with:

- **index.html**: Main structure and content of the portfolio.
- **style.css**: Modern dark-themed styling with gradient accents and animations.
- **script.js**: Interactive elements including typing effects, smooth scrolling, and form handling.
- **assets/**:
  - **images/**: Directory for profile photos and project images.
  - **resume/**: Directory for the downloadable resume PDF.
- **.git/**: Version control history.
